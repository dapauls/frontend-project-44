# Project description:
 "Brain-games" - these are five educational mini-games (Prime Game, Even Game, GCD Game, Progression Game, Calculator Game).
## Installing the game: 
Minimum requirements: 

```Node.js 13+```
<br/>

Installation:

```
git@github.com:dapauls/frontend-project-44.git 
make install
make lint 
```
